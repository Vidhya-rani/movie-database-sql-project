A SQL-based database project designed to manage movies, actors, and their relationships using MySQL.

## Tech Stack
- SQL
- MySQL

## Database Design
- Designed normalized relational tables for movies and actors
- Implemented many-to-many relationships using a junction (mapping) table
- Enforced data integrity using primary and foreign keys

## Features
- Store movie details (title, genre, release year, rating)
- Store actor and actress information
- Maintain relationships between movies and actors
- Perform full CRUD operations on all entities

## SQL Queries Implemented
- Fetch all actors associated with a specific movie
- Retrieve all movies acted by a specific actor
- Display top-rated movies
- Identify movies with more than one actor
- Execute complex JOIN queries across multiple tables
